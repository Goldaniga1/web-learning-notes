SDG11 進場動畫的完整流程：
1. HTML 元素加上 class="fade-up"（初始透明 + 往下偏移）
2. IntersectionObserver 監視這些元素
3. 使用者捲動到卡片時 → 加上 .visible class
4. CSS transition 讓 opacity 和 transform 平滑過渡 → 看起來像「飛進來」
