06 — JavaScript DOM

用 JS 操控 HTML / CSS · SDG11 所有互動的核心

DOM 是什麼

DOM（Document Object Model）是瀏覽器把 HTML 轉換成的樹狀結構。JavaScript 透過操作 DOM 來改變頁面。

比喻：HTML 是蓋好的房子，DOM 是房子的平面圖（你可以看到每個房間）。JS 就是你手裡的改建工具，可以加房間、改顏色、拆牆。

選取 HTML 元素

// 用 id 選（最常用） const modal = document.getElementById('scene-modal'); // 用 CSS 選擇器選（一個） const btn = document.querySelector('.lang-btn'); const h1 = document.querySelector('#hero h1'); // 用 CSS 選擇器選（所有符合的） const allCards = document.querySelectorAll('.tw-city-card'); const allI18nEl = document.querySelectorAll('[data-en]'); // ^ 選所有有 data-en 屬性的元素（SDG11 語言切換就這樣選） // 跑過每一個 allCards.forEach(card => { console.log(card); });

修改元素的內容和樣式

const el = document.getElementById('my-div'); // 改文字（安全，不解析 HTML） el.textContent = '新的文字'; // 改 HTML 內容（可以放標籤，但小心 XSS） el.innerHTML = '<strong>粗體</strong>'; // 改 CSS 樣式 el.style.color = '#f99d26'; el.style.fontSize = '1.5rem'; el.style.display = 'none'; // 隱藏 el.style.width = '55%'; // SDG11 進度條 // 讀取屬性（data- 屬性） const enText = el.dataset.en; // 讀 data-en 的值 const value = el.dataset.value; // 讀 data-value（進度條數值）

classList — 新增/移除 CSS class

這是 SDG11 最常用的技巧。改 class 讓 CSS 自動處理樣式，比直接改 style 更乾淨。

const modal = document.getElementById('scene-modal'); // 加 class modal.classList.add('active'); // 現在 modal 有 class="active" // 移除 class modal.classList.remove('active'); // 移除 active // 切換（有就移除，沒有就加） modal.classList.toggle('open'); // 檢查有沒有這個 class if (modal.classList.contains('active')) { console.log('Modal 是開著的'); } // SDG11 openCity() 的核心： function openCity(name) { // ... 設定內容 ... document.getElementById('scene-modal').classList.add('active'); // CSS: #scene-modal.active { display: flex; } }

Modal 怎麼做的（SDG11 核心）

/* CSS */ #my-modal { display: none; /* 預設隱藏 */ position: fixed; inset: 0; background: rgba(0,0,0,0.8); align-items: center; justify-content: center; } #my-modal.active { display: flex; /* 加了 active class 才顯示 */ } /* JavaScript */ function openModal() { document.getElementById('my-modal').classList.add('active'); } function closeModal() { document.getElementById('my-modal').classList.remove('active'); }

IntersectionObserver — 滾到哪裡才出現

SDG11 的卡片進場動畫（fade-up）就是靠這個。監聽元素有沒有進入視窗。

// 建立觀察器 const observer = new IntersectionObserver((entries) => { entries.forEach(entry => { if (entry.isIntersecting) { // 進入視窗了！ entry.target.classList.add('visible'); // CSS: .fade-up.visible { opacity: 1; transform: translateY(0); } } }); }); // 觀察所有有 .fade-up 的元素 document.querySelectorAll('.fade-up').forEach(el => { observer.observe(el); });

SDG11 語言切換的完整原理

/* HTML 上面存好兩種語言 */ <span data-en="Statistics" data-zh="統計數據">Statistics</span> /* JavaScript 切換時 */ let currentLang = 'en'; function toggleLang() { // 1. 切換語言狀態 currentLang = currentLang === 'en' ? 'zh' : 'en'; // 2. 更新按鈕文字 document.querySelector('.lang-btn').textContent = currentLang === 'en' ? '中文' : 'English'; // 3. 選所有有 data-en 屬性的元素，換文字 document.querySelectorAll('[data-en]').forEach(el => { el.textContent = el.dataset[currentLang]; // el.dataset['en'] 或 el.dataset['zh'] }); }
