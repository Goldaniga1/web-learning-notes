📌 SDG11 進度條動畫原理：
1. HTML 用 data-value="55" 存數值
2. CSS 設 transition: width 1.3s cubic-bezier(0.4, 0, 0.2, 1)
3. JS 偵測到元素進入視窗時，設 style.width = '55%'
4. width 從 0% 平滑動到 55%，動畫效果就出來了
