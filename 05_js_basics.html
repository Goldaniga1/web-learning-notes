05 — JavaScript 基礎

變數 · 函式 · 條件 · 迴圈 · 陣列 · 物件

JavaScript 在哪裡寫

/* 方式 1：寫在 HTML 的 <script> 標籤裡（SDG11 就是這樣）*/ <script> console.log('Hello!'); </script> /* 方式 2：外部 .js 檔案 */ <script src="script.js"></script> /* 方式 3：HTML 行內（onclick 屬性）*/ <button onclick="toggleLang()">中文</button>

變數（Variable）— 儲存資料

// 三種宣告方式 let name = '台北'; // 可以重新賦值（推薦） const city = '高雄'; // 不可以重新賦值（推薦用於常數） var old = '舊寫法'; // 舊版，避免用 // 資料型別 let num = 42; // 數字（Number） let text = 'Hello'; // 字串（String） let flag = true; // 布林（Boolean）：true / false let nothing = null; // 空值 // SDG11 的例子 let currentLang = 'en'; // 現在是英文還是中文 let currentIdx = 0; // 場景 Modal 現在第幾張圖

函式（Function）— 可重複使用的程式碼

// 宣告函式 function greet(name) { console.log('Hello, ' + name); } greet('Jonas'); // 呼叫：Hello, Jonas // 有回傳值的函式 function add(a, b) { return a + b; } let result = add(3, 5); // result = 8 // SDG11 的函式 function toggleLang() { // 切換語言的邏輯... } function openCity(cityName) { // 根據 cityName 開啟對應城市的場景 Modal }

條件判斷（if / else）

let lang = 'en'; if (lang === 'en') { console.log('English mode'); } else { console.log('中文模式'); } // 三元運算子（簡短版的 if/else） let btnText = lang === 'en' ? '中文' : 'English'; // 條件 true時 false時 // SDG11 toggleLang 的核心邏輯 currentLang = currentLang === 'en' ? 'zh' : 'en';

陣列（Array）— 多筆資料的清單

const cities = ['台北', '高雄', '台中', '台南']; // 取值（從 0 開始數） console.log(cities[0]); // 台北 console.log(cities[2]); // 台中 console.log(cities.length); // 4（總共幾筆） // 用 forEach 跑每一筆 cities.forEach(function(city) { console.log(city); }); // SDG11 的城市照片就是陣列 const scenes = [ { title: '台北 101', photo: 'https://...' }, { title: '台北捷運', photo: 'https://...' }, ]; // scenes[0].title → '台北 101' // scenes[1].photo → 照片 URL

物件（Object）— 有名稱的資料組合

const taipei = { name: '台北', population: 2600000, recyclingRate: 55, isCapital: true, }; // 取值 console.log(taipei.name); // 台北 console.log(taipei['recyclingRate']); // 55（另一種寫法） // SDG11 城市查詢資料（物件裡包物件） const cityData = { taipei: { en: { name: 'Taipei', highlight: 'Pay-as-you-throw...' }, zh: { name: '台北', highlight: '垃圾費隨袋徵收...' } }, }; // cityData.taipei.en.name → 'Taipei'

迴圈（Loop）— 重複做同一件事

// for 迴圈 for (let i = 0; i < 5; i++) { console.log(i); // 0, 1, 2, 3, 4 } // 跑陣列 const cities = ['台北', '高雄', '台中']; for (let i = 0; i < cities.length; i++) { console.log(cities[i]); } // 更簡單的 forEach cities.forEach((city, index) => { console.log(index, city); });

Template Literals — 字串拼接

const city = '台北'; const rate = 55; // 舊方式（不好看） let msg1 = '城市：' + city + '，回收率：' + rate + '%'; // 新方式（用反引號 ` 和 ${} ） let msg2 = `城市：${city}，回收率：${rate}%`; // 結果一樣：城市：台北，回收率：55%

console.log — 除錯的好幫手

console.log('Hello'); // 輸出到 F12 的 Console console.log(myVariable); // 看變數的值 console.log('lang:', lang); // 帶說明
📌 按 F12 打開 DevTools，點 Console 分頁，就能看到 console.log 的輸出。這是 debug 最常用的工具。
