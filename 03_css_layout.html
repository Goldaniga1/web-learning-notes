display — 元素怎麼排列

值行為例子
block獨佔一整行，可設寬高div, p, h1, section
inline在行內排列，不能設寬高span, a, strong
inline-block在行內排列，但可設寬高圖片、按鈕
flex彈性盒子排版（現代首選）導覽列、卡片區
grid網格排版複雜二維佈局
none完全隱藏（不佔空間）隱藏 Modal

position — 脫離正常排版

值行為SDG11 用在
static預設，正常排版大部分元素
relative相對自己原本位置偏移，不脫離文件流做 absolute 的錨點
absolute相對最近的 positioned 父元素，脫離文件流Modal 關閉按鈕
fixed相對視窗固定，捲動不影響城市場景 Modal
sticky捲到指定位置時黏住導覽列（top:0）

📌 SDG11 排版總結：
· 導覽列：display: flex + justify-content: space-between
· 城市卡片、統計卡片：display: flex + flex-wrap: wrap + justify-content: center
· Modal 蓋住畫面：position: fixed + inset: 0 + display: flex
· 導覽列跟著捲動：position: sticky + top: 0
