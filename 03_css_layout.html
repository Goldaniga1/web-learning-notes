03 — CSS 排版

Flexbox · position · z-index · SDG11 排版全用這些

display — 元素怎麼排列

值行為例子
block獨佔一整行，可設寬高div, p, h1, section
inline在行內排列，不能設寬高span, a, strong
inline-block在行內排列，但可設寬高圖片、按鈕
flex彈性盒子排版（現代首選）導覽列、卡片區
grid網格排版複雜二維佈局
none完全隱藏（不佔空間）隱藏 Modal

Flexbox — 最重要的排版方式

在父元素加上 display: flex，子元素就會自動排在同一行。

/* 父元素：容器 */ .container { display: flex; /* 主軸方向（預設橫向）*/ flex-direction: row; /* 橫排（預設）*/ /* flex-direction: column; */ /* 直排 */ /* 主軸對齊（橫向時：左中右）*/ justify-content: center; /* 置中 */ /* justify-content: space-between; */ /* 兩端對齊 */ /* justify-content: flex-start; */ /* 靠左（預設）*/ /* 交叉軸對齊（橫向時：上中下）*/ align-items: center; /* 垂直置中（超常用！）*/ /* 間距 */ gap: 24px; /* 子元素間的距離 */ /* 換行（子元素太多時）*/ flex-wrap: wrap; /* 允許換行 */ }

子元素（flex item）屬性

/* 子元素屬性 */ .item { /* flex: 1 讓這個子元素填滿剩餘空間 */ flex: 1; /* 不讓子元素縮小 */ flex-shrink: 0; /* 覆蓋父元素的 align-items，對自己單獨設定 */ align-self: flex-end; }

position — 脫離正常排版

值行為SDG11 用在
static預設，正常排版大部分元素
relative相對自己原本的位置偏移，不脫離文件流做 absolute 的錨點
absolute相對最近的 position 非 static 的父元素，脫離文件流Modal 內的關閉按鈕
fixed相對視窗固定，捲動不影響城市場景 Modal（inset:0）
sticky捲到指定位置時固定導覽列（top:0）
/* SDG11 導覽列：sticky */ nav { position: sticky; top: 0; /* 捲到頂部就黏住 */ z-index: 100; /* 確保在其他元素上面 */ } /* SDG11 城市場景 Modal：fixed + inset */ #scene-modal { position: fixed; inset: 0; /* 等於 top:0 right:0 bottom:0 left:0 */ /* → 蓋住整個視窗 */ } /* Modal 裡的關閉按鈕：absolute */ .scene-close-btn { position: absolute; top: 24px; right: 32px; /* 右上角 */ }

z-index — 圖層順序

決定哪個元素在上面（只在 position 非 static 時有效）。

/* 數字越大，越在上面 */ nav { z-index: 100; } /* 導覽列 */ #scene-modal { z-index: 999; } /* 場景 Modal（蓋住導覽列）*/ #sources-modal { z-index: 1000; } /* 資料來源（最上層）*/

水平置中技巧

/* 方法1：margin: 0 auto（block 元素置中）*/ .container { max-width: 900px; margin: 0 auto; /* 左右自動，等於水平置中 */ } /* 方法2：flexbox（最常用）*/ .parent { display: flex; justify-content: center; align-items: center; } /* 方法3：垂直水平置中（Modal）*/ .modal { position: fixed; inset: 0; display: flex; align-items: center; justify-content: center; }

overflow — 超出範圍怎麼辦

body { overflow-x: hidden; } /* 禁止橫向捲動（SDG11 有用）*/ /* 手機版表格橫向捲動 */ .cities-table-wrap { overflow-x: auto; /* 內容超出時出現橫向捲軸 */ -webkit-overflow-scrolling: touch; /* iOS 順滑捲動 */ } /* 讓圓角正常裁切 */ .card { border-radius: 14px; overflow: hidden; /* 防止子元素超出圓角 */ }
📌 SDG11 排版總結：
· 導覽列：display: flex + justify-content: space-between
· 城市卡片、統計卡片：display: flex + flex-wrap: wrap + justify-content: center
· Modal 蓋住畫面：position: fixed + inset: 0 + display: flex
· 導覽列跟著捲動：position: sticky + top: 0
