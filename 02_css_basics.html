02 — CSS 基礎

外觀、顏色、大小、字體 · 讓頁面好看的語言

CSS 怎麼寫

CSS 有三種寫法：

/* 1. 外部檔案（最推薦，寫在 .css 檔案裡）*/ <link rel="stylesheet" href="style.css"> /* 2. 內部樣式（寫在 <head> 的 <style> 裡，SDG11 就是這樣）*/ <style> h1 { color: red; } </style> /* 3. 行內樣式（直接寫在標籤上，不推薦大量使用）*/ <p style="color: red; font-size: 18px;">文字</p>

CSS 語法結構

/* 選擇器 { 屬性: 值; 屬性: 值; } */ h1 { color: #f99d26; /* 文字顏色 */ font-size: 2rem; /* 字體大小 */ font-weight: bold; /* 粗體 */ } /* 多個選擇器可以共用同一組樣式 */ h1, h2, h3 { font-family: 'Segoe UI', sans-serif; }

選擇器（Selector）— 選哪個元素

選擇器選到誰範例
*所有元素* { margin: 0; }
標籤所有該標籤p { color: gray; }
.class有這個 class 的元素.card { border-radius: 12px; }
#id有這個 id 的元素（唯一）#hero { height: 100vh; }
A BA 裡面的所有 B（後代）nav a { color: white; }
A > BA 的直接子元素 Bul > li { list-style: none; }
A:hover滑鼠移到 A 上.card:hover { transform: scale(1.05); }
A::beforeA 的前面插入內容h2::after { content: ''; display: block; }
/* SDG11 實際用到的選擇器範例 */ nav ul li a { /* nav 裡的 ul 裡的 li 裡的 a */ color: white; text-decoration: none; } .tw-city-card:hover { /* 滑鼠移到城市卡片上 */ transform: translateY(-7px); box-shadow: 0 14px 40px rgba(249,157,38,0.18); } section h2::after { /* 每個 section 標題下方的橘色底線 */ content: ''; display: block; width: 56px; height: 4px; background: #f99d26; }

Box Model — 每個元素都是一個盒子

這是 CSS 最重要的概念之一。每個 HTML 元素都由四層組成：

LIVE DEMO — Box Model
margin（外距，與其他元素的距離）
border（邊框）
padding（內距，內容與邊框的距離）
content（實際內容）
.card { /* 內距：上下 32px，左右 24px */ padding: 32px 24px; /* 外距：四邊都是 16px */ margin: 16px; /* 也可以分開寫 */ margin-top: 20px; padding-left: 24px; /* 邊框 */ border: 1px solid #ccc; /* 粗細 樣式 顏色 */ border-radius: 12px; /* 圓角 */ /* box-sizing: border-box 讓 width 包含 padding 和 border */ /* SDG11 用 * { box-sizing: border-box } 全部套用 */ }

字體相關屬性

p { font-size: 1rem; /* 字體大小（rem 相對於根元素，通常 1rem = 16px）*/ font-weight: 700; /* 粗細：100(細) ~ 900(粗)，bold = 700 */ font-family: 'Segoe UI', Arial, sans-serif; /* 字型，後面是備用 */ line-height: 1.7; /* 行高（1.7 = 字體大小的 1.7 倍）*/ letter-spacing: 2px; /* 字元間距 */ text-align: center; /* 對齊：left / center / right */ text-decoration: none; /* 去掉連結的底線 */ text-transform: uppercase; /* 轉大寫 */ white-space: nowrap; /* 不換行 */ opacity: 0.7; /* 整體透明度 0~1 */ }

CSS 單位

單位意義適合用在
px像素，固定大小邊框、細部尺寸
rem相對於根元素字體（通常 1rem = 16px）字體大小、間距（推薦）
em相對於父元素字體字體、padding
%相對於父元素大小寬度、高度
vw視窗寬度的 1%全螢幕寬度
vh視窗高度的 1%全螢幕高度（100vh）

CSS 變數（SDG11 有用到）

把常用的值存起來，改一個地方全部都變。

/* 在 :root 定義（全域可用）*/ :root { --orange: #f99d26; --dark-orange: #e07b00; } /* 使用 */ .btn { background: var(--orange); border: 2px solid var(--dark-orange); } /* 好處：想改主題色只改 :root 一個地方就好 */

選擇器優先級（Specificity）

當多個選擇器都選到同一個元素，優先級高的會覆蓋低的：

選擇器優先級
* （全選）最低
p（標籤）低
.class（類別）中
#id高
style=""（行內）最高
!important核武（不到萬不得已不要用）
常見錯誤：寫了 CSS 但沒效果，常常是因為被更高優先級的選擇器覆蓋了。用 F12 DevTools 可以看到哪個規則被劃掉（strikethrough）。
