選擇器優先級（Specificity）

當多個選擇器都選到同一個元素，優先級高的會覆蓋低的：

選擇器優先級（高→低）
* （全選）最低
p（標籤）
.class（類別）
#id
style=""（行內）最高
!important核武（不到萬不得已不要用）
看哪個規則贏
預覽
除錯技巧：CSS 寫了但沒效果，最常見原因是被更高優先級的選擇器覆蓋。按 F12 開 DevTools → Elements → Styles，被覆蓋的規則會顯示刪除線。
