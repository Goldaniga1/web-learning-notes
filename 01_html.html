HTML 是什麼？

HTML（HyperText Markup Language）是用來描述網頁內容結構的語言。它告訴瀏覽器「這是標題」「這是段落」「這是圖片」，但不管顏色或動畫——那是 CSS 和 JavaScript 的事。

🏠 比喻：HTML = 房子的牆壁和樓板（結構）／CSS = 油漆和裝潢（外觀）／JS = 電燈開關和電梯（互動）

語意元素（Semantic HTML）— 點各區塊看說明

下面是一個典型網頁的結構，點擊每個區塊查看它的用途。

<header> 頁首：網站 Logo、標語
通常放網站 Logo、標語，有時候包含 <nav>。SDG 11 的橘色導覽列就在 header 裡。
<nav> 導覽列：連結選單
放導覽連結（Statistics / Taiwan Focus / Goals...）。用 <ul> + <li> + <a> 組成。螢幕閱讀器會特別識別這個標籤。
<main> 主要內容（一頁只能有一個）
整個頁面最核心的內容放這裡。一個 HTML 文件只能有一個 <main>，搜尋引擎會特別重視這裡的內容。
<section>   章節：有主題的獨立區塊
SDG 11 有 #stats、#taiwan、#goals... 等多個 section。每個 section 通常有自己的 <h2> 標題。
<section>   另一個章節
section 可以有很多個，它們會依序排列。加上 id 屬性就可以做錨點跳轉（href="#stats"）。
<footer> 頁尾：版權、聯絡資訊
網頁最下方。通常放版權聲明、作者、聯絡資訊、資料來源等。
語意 vs div：技術上 <div class="nav"> 也能做到一樣的效果，但用 <nav> 讓程式碼更清楚、搜尋引擎更好理解、無障礙更好。
